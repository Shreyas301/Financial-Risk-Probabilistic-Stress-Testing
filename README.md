# Financial-Risk-Probabilistic-Stress-Testing

## 📌 Project Overview
This repository contains a high-precision framework for evaluating credit risk and performing portfolio stress tests using Monte Carlo simulations. The project demonstrates how **algorithmic weight rebalancing** can proactively mitigate exposure to extreme market events, moving beyond static traditional scoring models.

[cite_start]**Key Result:** Successfully identified and implemented a **15% reduction in Value at Risk (VaR)** through automated rebalancing logic[cite: 61, 112].

## 🛠️ Tech Stack
* [cite_start]**Language:** Python 3.9+ [cite: 19, 71]
* [cite_start]**Libraries:** NumPy, Pandas, SciPy, Matplotlib [cite: 13, 71]
* [cite_start]**Methodologies:** Monte Carlo Simulation, Inverse-Variance Optimization, Probabilistic Stress Testing [cite: 20, 60, 111]

## 🚀 Key Features
* [cite_start]**Credit Risk Scoring**: Statistical validation of asset risk profiles using Gaussian distributions[cite: 20, 72].
* [cite_start]**Monte Carlo Engine**: Simulates 10,000+ market iterations to forecast end-of-year portfolio values[cite: 60, 111].
* [cite_start]**Algorithmic Rebalancing**: Automatically shifts portfolio weights from high-volatility to low-volatility assets to optimize stability[cite: 61, 112].
* [cite_start]**Visual Analytics**: Dynamic histograms displaying the shift in VaR thresholds between baseline and optimized states[cite: 12, 109].

## 📊 Methodology & Outcomes
### 1. Motivation
In the Fintech sector, standard risk assessments often fail to account for "tail risks." [cite_start]This project creates a dynamic framework to simulate these rare but impactful events[cite: 9, 82].

### 2. Execution
1.  [cite_start]**Baseline Calculation**: Establishing the initial VaR using equal asset weighting[cite: 68, 99].
2.  [cite_start]**Stress Testing**: Running probabilistic simulations to identify system bottlenecks[cite: 53, 55].
3.  [cite_start]**Optimization**: Applying an inverse-variance algorithm to redistribute weights[cite: 61, 112].

### 3. Results
* [cite_start]**Before Optimization**: Standard exposure to market volatility[cite: 111].
* [cite_start]**After Optimization**: Achieved a **15% VaR reduction**, significantly enhancing portfolio resilience[cite: 61, 112].
